# TinyAcc — Multi Game Instance

Run more than one game client at the same time.

## Download

[**TinyAcc.exe**](https://github.com/VirageRoblox/TinyAcc/releases/latest/download/TinyAcc.exe) — one file, no install.

## Use

1. Close Roblox completely.
2. Open TinyAcc (it shows **Multi-instance: ON**).
3. Launch Roblox as many times as you want.

Keep the window open. Close it to turn off.

If TinyAcc warns that the game is already open, close every game window and click **Retry**.

*Windows may say "unknown publisher" — click More info → Run anyway.*
