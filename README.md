# Roblox Multi-Instance

Run more than one Roblox at the same time.

## Download

[**RobloxMultiInstance.exe**](https://github.com/VirageRoblox/RobloxMultiInstance/releases/latest/download/RobloxMultiInstance.exe) — one file, no install.

## Use

1. Close Roblox.
2. Open the app (it shows **On**).
3. Launch Roblox as many times as you want.

Keep the window open. Close it to turn off.

*Windows may say "unknown publisher" — click More info → Run anyway.*
